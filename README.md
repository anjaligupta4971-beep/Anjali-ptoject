# Portfolio Website

A modern, responsive 4-page HTML website showcasing projects and code examples for an intern portfolio.

## Pages

1. **index.html** - Home page with hero section, features, and quick links
2. **about.html** - About page with personal information, skills, and education timeline
3. **projects.html** - Projects page with detailed code examples including:
   - Interactive Calculator
   - Todo List Application
   - Responsive Navigation Menu
   - Form Validation
4. **contact.html** - Contact page with contact information and a contact form

## Features

- ✅ Fully responsive design (mobile-friendly)
- ✅ Modern and clean UI/UX
- ✅ Code examples with syntax highlighting
- ✅ Smooth navigation between pages
- ✅ Professional color scheme
- ✅ Sticky navigation bar
- ✅ Interactive elements and hover effects

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (for form handling and interactivity)

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate between pages using the navigation menu
3. View code examples on the Projects page
4. Customize the content, colors, and styling to match your needs

## Customization

- Update personal information in `about.html` and `contact.html`
- Modify project descriptions and code examples in `projects.html`
- Change colors in `styles.css` (look for color values like `#3498db`, `#2c3e50`)
- Add your own projects and code snippets
